# Python Project for Data Science
## About
This project extracts real stock data and revenue data for Tesla and GameStop,
then visualizes them using Python.

## What This Project Does
- Extracts Tesla and GameStop historical stock data using yfinance
- Scrapes quarterly revenue data from web pages using BeautifulSoup
- Cleans and processes the data using Pandas
- Builds a dashboard with stock price and revenue graphs

## Tools and Libraries Used
- Python
- yfinance
- BeautifulSoup
- Pandas
- Matplotlib
- Requests
- NumPy

## Skills Demonstrated
- Web scraping
- API data extraction
- Data cleaning
- Data visualization
